## Overview

learn Documentation provides a centralized hub for your project's knowledge base. You create, organize, and share documentation effortlessly, enabling teams to collaborate in real-time. This platform supports MDX for rich, interactive pages with components like steps, tabs, and code examples.

## What is learn Documentation

learn Documentation serves as your default space to manage all project docs. You build structured pages using MDX syntax, incorporating components for engaging content. It integrates seamlessly with your workflow, supporting version control and search.

## Key Benefits for Teams

You gain several advantages from using learn Documentation:

- **Collaboration**: Multiple users edit simultaneously with live previews.
- **Customization**: Tailor pages with brand colors like `#3B82F6` and custom icons.
- **Scalability**: Organize docs into spaces, collections, and pages hierarchically.

## Getting Oriented with the Platform

Follow these steps to begin:

<Steps>
  <Step title="Create Your Space" icon="folder-plus">
    Navigate to your dashboard and select "New Space". Name it after your project, like "MyApp Docs".
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Inside the space, click "New Page" and choose the `introduction.mdx` template.
  </Step>
  <Step title="Customize with Components" icon="pencil">
    Edit the MDX content. Add `<Columns>` for layouts or `<CodeGroup>` for examples.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Save changes and share the public URL with your team.
  </Step>
</Steps>
