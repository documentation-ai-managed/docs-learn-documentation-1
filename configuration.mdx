## Applying Brand Colors and Themes

Customize your documentation's appearance to reflect your identity. Update the primary color, logos, and fonts in the workspace settings.

<Steps>
  <Step title="Access Settings" icon="settings">
    Navigate to **Workspace > Settings** in your dashboard.
  </Step>
  <Step title="Update Brand Color" icon="palette">
    Enter your hex code, such as `#3B82F6`, in the **Brand Color** field.
  </Step>
  <Step title="Apply Theme" icon="save">
    Select a theme preset and click **Apply Changes**. Preview updates live.
  </Step>
</Steps>

<Callout kind="tip">
  Test your `{brandColor}` across light and dark modes to ensure readability.
</Callout>

For advanced customization, edit the config file:

<CodeGroup tabs="JSON,YAML">
  ```json
  {
    "branding": {
      "primaryColor": "#3B82F6",
      "logoUrl": "https://your-domain.com/logo.svg",
      "fontFamily": "Inter, sans-serif"
    }
  }
  ```
  ```yaml
  branding:
    primaryColor: "#3B82F6"
    logoUrl: https://your-domain.com/logo.svg
    fontFamily: Inter, sans-serif
  ```
</CodeGroup>

## Workspace Permissions

Manage who can view, edit, or administer your documentation. Assign roles at the workspace or document level.

| Role          | View | Edit | Admin | Description                          |
|---------------|------|------|-------|--------------------------------------|
| Viewer        | ✅   | ❌   | ❌    | Read-only access to published docs. |
| Editor        | ✅   | ✅   | ❌    | Edit content and settings.          |
| Admin         | ✅   | ✅   | ✅    | Full control, including users.      |

<Expandable title="Advanced Permission Rules" default-open="false">
  Use environment variables for granular control:

  ```bash
  export LEARN_EDITOR_GROUPS="team:developers,team:design"
  export LEARN_VIEWER_ACCESS="public"
  ```

  Restart your workspace after changes.
</Expandable>

## Export and Import Settings

Backup or migrate your content easily. Export to Markdown, PDF, or JSON; import from ZIP archives.

1. Go to **Workspace > Export**.
2. Select format and download.
3. For import, upload files via **Import** tab.

<Callout kind="info">
  Exports include all versions and attachments up to 500MB.
</Callout>
